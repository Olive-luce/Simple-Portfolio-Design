## Retro Portfolio Website

A lightweight, retro-style personal portfolio built using HTML and CSS, inspired by classic operating systems and early desktop interfaces. The design uses pixel-style typography, window-based UI components, and a minimal layout to create a nostalgic yet modern web experience.
